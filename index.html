How Many Days Will My A/B Test Take?

Step 1. Collect Existing Data

The number of pageviews received by the page you plan to test on over the last month. Get this data in your Google Analytics account (show me how).
The number of conversions which occured from that page in the last month. Get this data from your website's admin dashboard or from your database.

Step 2. Results!

The number of days that your test takes depends on the impact of the change you make.

If the change increases conversion rate by only 1%, you will have to run the test for longer than if it incrases rates by 10%.

Getting the pageviews

Open up your Google Analytics. In the sidebar, click "Behavior", then "Content" and finally "All Pages".

Use the search box filter results until you find the page you plan to run the A/B test on. Take note of the number of pageviews that page has. Enter that value into the field below.
